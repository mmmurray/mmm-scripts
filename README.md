# mmm-scripts

An opinionated tool for building personal projects.

## Scripts

- `jest` - Jest watch.
- `coverage` - Jest coverage.
- `build` - TypeScript build.
- `pack` - Webpack build.
- `format` - Prettier.,
- `watch` - Runs TypeScript watch, Webpack Watch, and nodemon `lib/server`.

## Inspiration
